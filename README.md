# wsl_bootstrap
Bootstrap ubuntu 18.04 under WSL for development

## Usage
    
##### Clone Repository

  `git clone https://github.com/paulalex/wsl_bootstrap.git`

##### Bootstrap WSL

`bash setup.sh`
